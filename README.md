# 🐱 GATO - AI Hand Gesture Game

An AI-powered gesture-controlled game built using Python, OpenCV, MediaPipe, and Pygame.

The game uses real-time hand tracking through webcam input to control gameplay mechanics using custom gesture recognition.

---

## 🚀 Features

- ✋ Real-time hand gesture detection
- 🎮 AI-powered gameplay interaction
- 📷 Computer vision using webcam
- 🐱 Animated cat character states
- 🔊 Sound effects and background music
- ⚔️ Enemy matching system
- 🧠 Gesture recognition using MediaPipe

---

## 🛠 Tech Stack

- Python
- Pygame
- OpenCV
- MediaPipe
- NumPy
